
## 项目名称

radare2增强

## 支持单位

龙芯中科技术股份有限公司

## 项目描述

radare2(r2)是一款反向工程工具，可实现多平台和跨平台支持，目前已初步支持在loongarch平台上进行分析，但缺少重定位信息，ir使用存在问题。项目目标是完善radare2平台支持，使其在LoongArch平台上更加成熟。
